Overview

Object detection using YOLOv8 (You Only Look Once). This project covers basic YOLO usage, training on custom datasets, and data augmentation techniques to improve model robustness.

YOLOv8 Bounding Boxes Data Augmentation Images + Annotations

Key Techniques

  • YOLOv8 Architecture - Single-shot detection with backbone, neck, and head
  • Bounding Box Prediction - Anchor-free detection with coordinate regression
  • Data Augmentation - Mosaic, mixup, HSV transforms, geometric transforms
  • Custom Training - Transfer learning from COCO weights to custom datasets
