Tool Registry

26 Tools for Recipe Management and Meal Planning

V1: Recipe Lookup (10 tools)

Read-only tools for searching and browsing recipes.

Tool Description
list_categories List all recipe categories
get_recipe_by_id Get recipe by ID
list_recipes List recipes, optionally by category
find_recipes_by_title Search by title substring (case-insensitive)
filter_recipes_by_cuisine Filter by cuisine type
get_recipe_ingredients Get ingredients for a recipe
get_ingredient_by_id Get ingredient details
find_ingredient_by_name Search ingredients by name
list_recipes_by_ingredient Find recipes using an ingredient
get_technique Get technique reference

V2: Meal Planning (+4 tools = 14 total)

Create meal plans and generate shopping lists.

Tool Description
scale_recipe_servings Scale ingredients to target servings
create_meal_plan Create meal plans with recipe entries
get_meal_plan Retrieve meal plan with recipe details
generate_shopping_list Aggregate ingredients from a meal plan

V3: Pantry & Discovery (+4 tools = 18 total)

Find recipes based on what you have and get substitutions.

Tool Description
find_recipes_by_pantry Find recipes using available ingredients
suggest_substitution Suggest ingredient substitutions
list_common_ingredients Most-used ingredients across recipes
find_similar_recipes Recipes with overlapping ingredients

V4: Favorites, History, Notes & Ratings (+8 tools = 26 total)

Track your cooking journey and personalize recipes.

Tool Description
add_favorite Mark a recipe as favorite
list_favorites List all favorite recipes
log_cook_event Record when a recipe was cooked
get_cook_history View cooking history
add_recipe_note Add personal note to a recipe
get_recipe_notes Get notes for a recipe
rate_recipe Rate a recipe (1-5 stars)
list_top_rated List highest-rated recipes

Usage Examples

# V1: Recipe Lookup categories = list_categories() recipes = find_recipes_by_title("tofu") ingredients = get_recipe_ingredients(5) # V2: Meal Planning scaled = scale_recipe_servings(5, 8) # Scale recipe 5 to 8 servings plan = create_meal_plan("Week of Jan 6", [1, 5, 12, 18]) shopping = generate_shopping_list(plan_id) # V3: Pantry Discovery can_make = find_recipes_by_pantry(["tofu", "rice", "soy sauce"]) sub = suggest_substitution("butter") # Returns: "coconut oil" # V4: Favorites & History add_favorite(5) log_cook_event(5, "2025-01-15") rate_recipe(5, 5) # 5 stars! add_recipe_note(5, "Added extra garlic, delicious!")
