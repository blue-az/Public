Domain Features

📖 Recipe Lookup

Search recipes by title, category, cuisine, or ingredients. Case-insensitive with deterministic results.

📅 Meal Planning

Create weekly meal plans, scale servings, and generate shopping lists automatically.

🧂 Pantry Discovery

Find recipes using ingredients you have. Get substitution suggestions.

⭐ Favorites & History

Track cooking history, save favorites, add notes, and rate recipes.

Quick Start

# List all categories list_categories() # Search recipes by title find_recipes_by_title("fettuccine") # Get recipe with ingredients get_recipe_by_id(11) # Tempeh Katsu get_recipe_ingredients(11) # Find recipes using available ingredients find_recipes_by_pantry(["chickpea flour", "cumin"]) # Create a meal plan create_meal_plan("Week 1", [1, 7, 11, 13, 15])
