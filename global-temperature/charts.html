Chart Generation Tools

These visualizations are generated by the 9 tools in GlobalTemperature variation_8:

Tool Description Data Source
plot_temperature_trend Temperature anomaly time series with Paris targets NASA GISTEMP
plot_co2_trend CO2 concentration (Keeling curve) NOAA Mauna Loa
plot_warming_stripes Ed Hawkins-style warming stripes NASA GISTEMP
plot_multi_indicator 4-panel climate indicators (full_range option) Multiple
plot_sea_level_trend Sea level rise with 2100 projection NOAA LSA
plot_emissions_comparison Top emitters bar chart OWID/GCP
plot_arctic_amplification Arctic vs global warming comparison GISTEMP Zonal
plot_ocean_heat_trend Ocean heat content (0-700m or 0-2000m) NOAA NCEI
plot_climate_dashboard 6-panel comprehensive dashboard Multiple

Usage Example

plot_multi_indicator({'full_range': True}) - Generate multi-indicator panel showing each dataset's complete available history.
