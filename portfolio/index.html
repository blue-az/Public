Domain Overview

Portfolio Agent is a comprehensive financial portfolio analysis system built on the Project Phoenix framework. It provides intelligent agent capabilities for holdings analysis, tax optimization, options trading, risk assessment, and investment recommendations.

Data is extracted from Morgan Stanley client statements with multi-snapshot support for historical tracking and trend analysis.
