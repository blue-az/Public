Domain Adapter Architecture

Each domain has a dedicated adapter providing a uniform interface regardless of internal implementation.

# BaseDomainAdapter Interface class BaseDomainAdapter(ABC): def _ensure_initialized(self): """Lazy load domain components""" def get_tools(self) -> List[StructuredTool]: """LangChain tools for direct mode (649+ tools)""" def get_meta_tool(self) -> StructuredTool: """Single aggregated tool for meta mode (14 tools)""" def execute(self, query: str) -> str: """Execute natural language query""" # Two Registry Patterns Supported # Dataclass-based: Tool(name=..., execute_fn=...) # Used by: TennisAgent, Portfolio # Dict-based: {'name': ..., 'handler': ...} # Used by: WQ, Optiver, Stan, AI_WQ, YieldModel, QCi, Parable, PPR
