Variation 1: Core (8 tools)

Product Discovery

Core tools for searching, comparing, and understanding QCi's product portfolio and industry solutions.

Tools

  • search_products - Search products by name, category, or keyword
  • get_product_details - Get detailed specifications
  • get_solutions - Get industry solutions
  • match_use_case - Find matching use cases
  • compare_products - Compare products side by side
  • recommend_product - Get product recommendations
  • get_customer_stories - Customer success stories
  • get_technologies - Technology information

Use Cases

  • Initial product research
  • Basic customer inquiries
  • Product comparison for prospects
  • Industry solution matching

Variation 2: Extended (14 tools)

Financial Analysis

All V1 tools plus financial and performance analysis capabilities for deeper sales engagement.

V1 Core Tools (8)

All tools from V1 included

V2 Extended Tools (6)

  • analyze_roi - Calculate return on investment
  • calculate_tco - Total cost of ownership analysis
  • simulate_performance - Simulate quantum performance
  • benchmark_products - Benchmark against classical solutions
  • generate_proposal - Generate sales proposals
  • predict_scaling - Performance scaling prediction

Use Cases

  • Sales engagement and proposal generation
  • Financial justification for quantum adoption
  • Performance benchmarking and simulation
  • TCO and ROI analysis for prospects

Variation 3: Complete (21 tools)

Interview Preparation

Complete tool set including interview preparation for sales roles at QCi.

V1 + V2 Tools (14)

All tools from V1 and V2 included

V3 Interview Tools (7)

  • analyze_job_requirements - Parse job requirements
  • map_experience_to_requirements - Skill mapping
  • identify_skill_gaps - Gap analysis with resources
  • generate_talking_points - STAR stories generation
  • prepare_interview_questions - Questions to ask interviewer
  • generate_preparation_report - Comprehensive prep report
  • run_preparation_workflow - Multi-step prep workflow

Use Cases

  • Interview preparation for QCi positions
  • Experience-to-requirements mapping
  • Talking points and STAR story generation
  • Comprehensive preparation workflows

Interfaces

CLI Interface

cd domains/QCiAgent/cli python main.py

Interactive command-line interface for natural language queries.

Agentic Cockpit (GUI)

cd domains/QCiAgent/cockpit_poc python qci_agent_cockpit.py

Tkinter-based Glass Box AI interface with plan preview and execution trace.

Cockpit Features

  • Natural language query input
  • Execution plan preview (Blueprint Panel)
  • Real-time execution trace
  • Result inspection and export
