Hybrid Overview

The Hybrid mode estimates line yield from category complexity, annual implant volume, and manufacturer portfolio mix. It is anchored for realistic CRM ranges: ICD around 90% and top pacemaker lines near 98%.

Companies

ABT, BSX, MDT

Line Categories

Pacemaker, ICD, CRT-D, CRT-P, TPS

Model

Deterministic calibrated hybrid scoring

Hybrid Tools

Tool Description Parameters
predict_hybrid_yield_by_company_line Predict yield for a single company + category line. company, device_category, year, maturity_bonus
rank_hybrid_lines Rank company-category lines by estimated yield. year, top_n, company, maturity_bonus

Model Equation

Yield = Base(category) + VolumeGain(category) * ln(annual_implants / 200k + 1) - 2.0 * (manufacturer_complexity_index - 1.10) + maturity_bonus

Outputs are clipped to realistic manufacturing bounds for this domain.
