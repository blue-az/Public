Domain Overview

Semiconductors is a deterministic market-intelligence domain modeled after PPR_Agent architecture and adapted for CPU, GPU, and foundry analysis.

It combines structured market-share snapshots, vendor fundamentals, daily pricing, and event timelines to support reproducible analyst workflows.

Phoenix Principles Applied

Write-Then-Verify

Variation 5/6 workflows persist catalyst notes and SOP runs, then explicitly verify artifacts.

Abstraction Ladder

Variations progress from basic lookup to trend/CI analysis, then SOP orchestration and visualization synthesis.

Domain Doctor Tools

Tools are narrow and deterministic: each function owns one analytical operation over SQLite state.

Anti-Hallucination Templates

Catalyst notes and interactive research pack outputs use structured templates and stable fields.

Test-and-Prove

Domain includes smoke tests across Variations 1-8 and generated artifacts for visual verification.
